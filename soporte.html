Reportar Bugs y Errores

Si experimentas crashes, errores o comportamiento inesperado, sigue este procedimiento:

1
Recolectar Logs

Localiza y comprime la carpeta /logs generada por la aplicación

2
Capturar Evidencia

Toma capturas de pantalla o graba un video mostrando el error

3
Enviar Reporte

Envía los archivos a sandulientos@protonmail.com con una descripción detallada

Preguntas Frecuentes

¿Qué requisitos de sistema tiene QuimiTech?

Windows x64 con .NET 9. Exclusivamente desarrollado para esta plataforma.

¿Cómo recibo las actualizaciones?

Las actualizaciones se entregan via MediaFire. Los usuarios con licencia de lanzamiento reciben actualizaciones gratuitas hasta el 5 de Mayo 2025.

¿El software es portable?

Actualmente requiere configuración manual. Se planea desarrollar un instalador automático en futuras versiones.

¿Qué hago si el software no inicia?

Verifica que tengas .NET 9 instalado y envía un reporte de error con los logs generados.
