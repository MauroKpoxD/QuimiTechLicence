Instalación y Requisitos

Requisitos mínimos:
  • Sistema Operativo: Windows 8, 10 o 11
  • Memoria RAM: 4 GB (8 GB recomendado)
  • Espacio en disco: 500 MB libres
  • .NET Framework: 4.8 o superior (incluido)
Recomendado: Windows 10/11, 8GB RAM, 1GB espacio libre.

No, está incluido. QuimiTech viene con todas las dependencias necesarias. Si tienes .NET Framework 4.8 o superior instalado, el software lo utilizará. Si no, incluye una versión portable que no requiere instalación.

Es un falso positivo común - El software es 100% seguro
¿Por qué sucede esto?

QuimiTech utiliza un sistema de licencias que necesita comunicación con el servidor para validar activaciones. Windows Defender es muy estricto con software que realiza conexiones de red, especialmente cuando es portable (.exe) y no está firmado digitalmente con certificado costoso.

¿Qué debo hacer?
1
Agregar excepción en Windows Defender
  1. Abre Seguridad de Windows
  2. Ve a Protección contra virus y amenazas
  3. Haz click en Configuración de protección contra virus y amenazas
  4. Desplázate a Exclusiones y haz click en Agregar o quitar exclusiones
  5. Agrega la carpeta donde está QuimiTech como excepción
2
Ejecutar como administrador (Solución rápida)
  • Haz click derecho sobre QuimiTech.exe
  • Selecciona "Ejecutar como administrador"
  • Confirma el aviso de Control de Cuentas de Usuario (UAC)
  • Esto generalmente resuelve el bloqueo inmediatamente
3
Verificar en Protección en tiempo real
  • Si Windows Defender aún lo bloquea temporalmente:
  • Ve a Historial de protección
  • Busca QuimiTech en "Elementos en cuarentena"
  • Selecciona "Permitir en el dispositivo"
Garantía de Seguridad: QuimiTech no contiene malware, virus, ni software malicioso. El código es revisado y solo realiza conexiones legítimas para validación de licencias. Tu privacidad y seguridad están garantizadas.
Detalles Técnicos:
  • Tipo de detección: "Comportamiento sospechoso" o "Aplicación no reconocida"
  • Razón: Software portable + conexiones de red + sin firma digital costosa
  • Conexiones: Solo a servidores legítimos de licencias QuimiTech
  • Frecuencia: Solo durante activación/validación de licencia
  • Datos enviados: Solo ID de licencia (ningún dato personal)
Si necesitas ayuda adicional con este proceso, contacta a soporte y te guiaremos paso a paso.

Licencias y Activación

La versión demo incluye todas las funciones por 30 días. Después de este período, el software se bloqueará completamente. No hay limitaciones en cuanto a funcionalidades durante el período de prueba.

Depende de tu licencia:
  • Demo: Ilimitado (solo 30 días)
  • 6 meses / 1 año: 1 dispositivo activo
  • ONE SHOT: 5 dispositivos diferentes
  • KIT Equipo: 20 licencias individuales
El software detecta cambios de hardware y puede requerir reactivación.

Contacta para reactivación. En caso de formateo o cambio de hardware:
  1. Envía un email a sandulientos827@protonmail.com
  2. Indica tu número de licencia o email de compra
  3. Solicita la reactivación
Las reactivaciones son gratuitas para licencias válidas.

Reportar un Error o Bug

1
Prepara la Información

Reúne todos los datos necesarios antes de reportar

  • Versión del software (v2.0.6)
  • Sistema operativo y versión
  • Descripción detallada del problema
  • Pasos para reproducir el error
2
Busca los Logs

Los archivos de registro ayudan a diagnosticar

Ubicación de logs: C:\Users\[TuUsuario]\AppData\Local\QuimiTech\logs\ O en la carpeta donde instalaste el software
3
Envía el Reporte

Usa el formato correcto para una respuesta rápida

Asunto: "Bug Report - QuimiTech v2.0.6"
Cuerpo debe incluir:
  • Descripción del problema
  • Pasos para reproducir
  • Comportamiento esperado vs actual
  • Capturas de pantalla (si aplica)
  • Logs adjuntos
